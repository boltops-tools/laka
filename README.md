## Usage

    laka hello yourname
    laka sub:goodbye yourname

The CLI tool also detects and tasks in the current folder's Rakefile and delegate to those tasks.

## Installation

Add this line to your application's Gemfile:

    gem "laka"

And then execute:

    bundle

Or install it yourself as:

    gem install laka
